# Webcomponents

We are currently working on webcomponents [@labir/embed](/webcomponents) integration based on [Lit](https://lit.dev/). The work is very much in progress.

<script
  type="module"
  src="https://cdn.jsdelivr.net/npm/@labir/embed/dist/embed.min.js"
  async
></script>

## A complex app

We work on a single HTML Element that will render a complex analytical application using webcomponents:

```html
<thermal-file-app url="/tucnaci_04.lrc"></thermal-file-app>
```

<thermal-manager>
  <thermal-registry uuid="tucnaks">
    <thermal-group>
      <thermal-file-app url="/tucnaci_04.lrc"></thermal-file-app>
    </thermal-group>
  </thermal-registry>
</thermal-manager>
