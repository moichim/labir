# Tailwind UI

The TailwindCSS integration is built on top of [@labir/react](/react-bridge). It stands on top of [@nextui-org/react](https://nextui.org/) library.

## Installation

Follow the installation instructions of [Tailwind](https://tailwindcss.com/docs/installation) and [NextUI](https://nextui.org/docs/guide/installation).

Do not forget to [hoist dependencies](https://nextui.org/docs/guide/installation#hoisted-dependencies-setup) if you use `pnpm` package manager.

### tailwind.config.js

Include @labir/tailwind and @nextui-org/react to Tailwind content setting. Your config should look like this:

```javascript filename="tailwind.config.js" {7-8,14}
const { nextui } = require("@nextui-org/react");

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    // ... your paths
    "../../node_modules/@labir/tailwind/**/*.{js,jsx,tsx}",
    "../../node_modules/@nextui-org/theme/dist/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [
    nextui()
  ],
};

```
